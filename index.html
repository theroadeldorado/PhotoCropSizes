For Designers

Use .svg over .png when possible

Use PhotoShop to resize/crop images to the correct sizes listed below. Then save for web File > Export > Save For Web. Use JPEG and check the progressive option.

The idea is to get the file size as small as possible.

compressor.io is a good tool for further compressing an image with almost no loss in quality.

For Non Designers

Use Fotor.com

  • On the top right you can import the photo you want to resize/crop.
  • On the left hand side you can click resize. Keep these locked so that you don't strech the image. Set the width or height to the match what the website requires. Click Apply.
  • Then crop the image to make the other dimmension match what the website needs.
  • Click Save in the top toolbar. You may have to create an account.
  • You can rename the file if needed. Select jpg in the image doesnt have transparentcy. If it does select png. Use High Quality. Then click Download.
  • Now use the newly created file to upload to the website.

    • compressor.io is a good tool for further compressing an image with almost no loss in quality.
